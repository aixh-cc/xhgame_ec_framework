# 第一个游戏-倒水游戏


## 游戏设计

有4个瓶子，其中1个空瓶,另外3个装满不同色的液体

【游戏玩法】：
点击一个瓶子选中，点击另一个瓶子，将选中瓶子的液体导入

【游戏胜利】：
所有瓶子都装上同色的液体

【游戏失败】：
规定操作次数下，未能达成目标。

## 抽象为数学问题

问：为什么要抽象成数学问题？

答：游戏开发第一部分，一般都先抽象为数学问题的吧。另外由数据驱动开发，减少重构。

### 数学问题
4个数组,每个数组代表一个瓶子，`[1,2,3]`代表瓶子里装了3色液体
`[1,2,3]` 往`[3,3]`瓶子里倒水,倒完后`[1,2,3]`,`[3,3,3]`
